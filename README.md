# Laboratory 2: Duo Student Portfolio with Daily Blog

## Project Overview
This project is a collaborative effort by **Vincent Valladolid** and **Vincent Andojar** for their **Web Systems and Technologies** course (BSIT 2B). The goal is to build a structured, professional portfolio website that showcases individual backgrounds, skills, and daily coding journeys through blog entries.

## Meet the Team
- **Vincent Valladolid** - BSIT 2B Student
- **Vincent Andojar** - BSIT 2B Student

## Laboratory Goals
- [x] Create a structured portfolio using HTML and CSS.
- [x] Collaborate effectively using Git and GitHub.
- [x] Demonstrate consistency through daily blog commits.
- [x] Maintain a clean and organized project repository.

## Features
- **Shared Landing Page:** A central hub to navigate between both students' portfolios.
- **Individual Portfolios:** Detailed sections for personal information, hobbies, interests, and career goals.
- **Daily Blogs:** A dedicated section for each student to post daily updates regarding their coding experiences, personal insights, and daily challenges.
- **Responsive Design:** A mobile-friendly layout ensuring a premium viewing experience across devices.
